# Reproducibility notes

This repository is an archived codebase derived from the original Jupyter
notebook workflow.

## Scope

The code documents the following stages:

1. Flood exposure construction.
2. Microdata sample construction.
3. Exposure linkage.
4. Fixed-effects estimation.
5. Mechanism analysis.
6. External flood dataset comparison.
7. Tables and figures.

## Notebook-export policy

Most scripts were generated by exporting code cells from original notebooks.
The exported scripts preserve the original workflow for auditability. They were
not refactored into a fully modular production pipeline.

## Local path policy

The original notebooks may contain local absolute paths. Each module also
includes `config.example.yaml` to document intended local inputs and outputs.
For public reuse, replace local paths with project-specific paths in a local
`config.yaml`.

## Data policy

Raw microdata, geospatial data, flood rasters, exposure-linked panels, and local
result outputs are not included.

## Suggested verification workflow

Before releasing the repository, confirm that only code, documentation, and
configuration examples are committed. Raw data, restricted microdata, derived
individual-level panels, local model outputs, plotting data, and final figures
should remain outside the public repository.
