# CHFS/CFHS household education mechanism code archive

This folder contains archived code for constructing household-level education
investment and financial mechanism variables from CHFS/CFHS notebooks.

The archive follows the user's existing Jupyter workflow and preserves original
notebook code cells for auditability.

## Data restrictions

Raw CHFS/CFHS microdata and household-level processed outputs are restricted and
are not included in this repository. Users must obtain the original data from
the official provider and comply with the data-use agreement.

## Repository use

This folder is intended to document the construction of household education
mechanism variables. It does not provide restricted data and should not be
interpreted as a fully self-contained replication package.
