# 2015 Census education fixed-effects regression code archive

This folder contains archived code for estimating the impact of childhood flood
exposure on education outcomes using 2015 Census microdata.

The archive follows the user's existing Jupyter workflow and preserves original
notebook code cells for auditability.

## Data restrictions

Raw 2015 Census microdata, individual-level analysis panels, and local regression
outputs are restricted and are not included in this repository.
