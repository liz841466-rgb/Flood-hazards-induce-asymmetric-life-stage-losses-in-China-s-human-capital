# External flood dataset comparison

This module archives code for comparing Gumbel-derived flood events with external
flood datasets and for using DFO as an alternative flood exposure dataset.

## Submodules

- `dfo_event_comparison/`: city- and county-level Gumbel flood events compared with DFO records.
- `gfd_event_validation/`: POD/FAR/CSI-style validation against external flood masks.
- `chinese_news_inventory/`: histogram scripts for Chinese news-based flood inventory validation.
- `dfo_alternative_exposure_results/`: DFO-based robustness analyses for child education and elderly health.
- `figures/`: DFO comparison and intensity-response figure scripts.

## Scope note

This module is not a validation of high-/low-risk zoning.
