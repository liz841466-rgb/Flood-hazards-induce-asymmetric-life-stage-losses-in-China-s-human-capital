# Child-education mechanism analysis run order

This folder archives the existing Jupyter workflow for household education investment, school facility supply, and schooling seasonality.

The scripts preserve notebook code cells. They are organized according to
the original workflow rather than refactored into a new production pipeline.

## Required workflow

1. `01_household_education_investment_mechanism.py`
   - Source notebook: internal original notebook (non-public source filename omitted).
   - Purpose: Household education investment mechanism

2. `02_school_facility_supply_mechanism.py`
   - Source notebook: internal original notebook (non-public source filename omitted).
   - Purpose: School facility supply mechanism

3. `03_monthly_schooling_seasonality.py`
   - Source notebook: internal original notebook (non-public source filename omitted).
   - Purpose: Monthly schooling seasonality mechanism

4. `04_spring_autumn_schooling_seasonality.py`
   - Source notebook: internal original notebook (non-public source filename omitted).
   - Purpose: Spring-autumn schooling seasonality mechanism

## Notes
