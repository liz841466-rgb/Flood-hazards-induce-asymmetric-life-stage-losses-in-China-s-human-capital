# Tables and figures

This module archives code for final figures, supplementary figures, EDA figures,
and external-flood supplementary figures.

The repository contains plotting code only. Figure outputs and local plotting
data are intentionally excluded.

## Submodules

- `main_figures/`: main Figure 2 and Figure 3 plotting notebooks.
- `supplementary_figures/`: supplementary figures for return levels, school terms, and status indices.
- `eda_figures/`: descriptive and sample-composition figures.
- `external_flood_figures/`: DFO/GFD supplementary figure code.
